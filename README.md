# Herramienta Administrativa de Base de Datos

una herramienta administrativa para bases de datos,
interactuando directamente con las tablas de sistema (system tables) del SGBD asignado. Esta
herramienta permite la administración de los principales objetos de la base de datos,
gestionando conexiones y ejecutando comandos SQL.
